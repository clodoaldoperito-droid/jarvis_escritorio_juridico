# ⚖️ JARVIS - ESCRITÓRIO JURÍDICO DIGITAL (V6.2)
## Alta Performance: Ambiental, Trabalhista e SST

Este repositório contém o motor de inteligência artificial da XTAL adaptado para o Direito Brasileiro, integrando perícia técnica de engenharia com estratégia jurídica avançada.

### 🪄 Como Iniciar (Mágica em 1 Clique):
1. Faça o download ou clone este repositório.
2. Certifique-se de ter o **Python 3.10+** instalado no seu Windows.
3. Clique duas vezes no arquivo **`MAGIC_START.bat`**.
4. O sistema irá configurar o ambiente e abrir o Dashboard no seu navegador automaticamente.

### 🔒 Modelo BYOK (Bring Your Own Key):
Este sistema é independente. Para funcionar, você precisará:
1. Abrir o arquivo **`.env`** (criado após o primeiro run).
2. Colocar sua própria chave de API (Google Gemini, OpenAI ou Anthropic).
3. O sistema usará os SEUS tokens para processar as petições e pareceres.

### 🧠 Células Disponíveis:
- **Ambiental:** Geotecnia, CETESB, CONAMA, Água, Ar e Solo.
- **Trabalhista/SST:** NRs, NHOs, Previdenciário (LTCAT/PPP) e eSocial.
